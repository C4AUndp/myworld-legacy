What is this?

The MY World ballot can easily be embedded as a widget on your own web site. This will allow visitors to your site to vote on MY World priorities without leaving your page.

How does it work?

To embed the widget on your page, use the simple <iframe> tag. Below is an example of how this code should look in your web page:

<iframe src="http://www.myworld2015.org/?iframe=600&partner=demo" width="600" height="800px" frameborder="0" style="border:1px solid #aaa;"></iframe>

Dimensions: 600 pixels wide, 500-800 pixels tall

The widget is designed to work at 600 pixels wide. Please only use this width when embedding on your site. You may set the height to whatever you think is appropriate, but a height of between 500 and 800 pixels is recommended.

Partner ID

Make sure to include your organization's partner ID in the frame's source URL. In the example above, replace partner=demo with partner=YourPartnerID.

Introductory Text

Because the widget contains minimal description of the MY World project, you are encouraged to include some introductory text on your web page above the widget. You should contextualize the ballot and promote your organization's involvement in the voting. For general information about the project, see the MY World about pages.

Language

To set the default language of your widget, include a lang=?? argument in the frame's source URL. The example below shows the URL for French:

http://www.myworld2015.org/?iframe=600&lang=fr

Available languages:

  • Arabic = ar
  • Chinese = ch
  • English = en
  • French = fr
  • Indonesian = bi
  • Korean = kr
  • Portuguese = pr
  • Russian = ru
  • Spanish = es
  • Thai = th
  • Vietnamese = vt
